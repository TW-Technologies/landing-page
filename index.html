Sobre Nós

Somos a equipe TWtechnologies , um grupo composto por cinco estudantes da FATEC de Registro, no curso de Desenvolvimento de Software Multiplataforma, e estamos unidos pelo desafio de um projeto acadêmico.

Nossa equipe foi formada com o objetivo de explorar, desenvolver e implementar soluções eficazes e criativas que possam fazer a diferença na vida das pessoas.

Nossa missão é desenvolver projetos inovadores que resolvam problemas reais, aplicando o conhecimento acadêmico de forma prática e colaborativa. Visamos criar soluções que não apenas cumpram os requisitos acadêmicos, mas que também tenham potencial para transformar positivamente a sociedade.

Projeto

O objetivo geral deste projeto chamado InfraCow é propor uma aplicação que por meio de uma câmera térmica receberá imagens de bovinos em prol de reconhecer uma eventual suspeita de DRB. Esta ferramenta utilizaria uma IA e a aplicação em Java para aprofundar a análise da doença, coletando informações de temperaturas por meio da termografia de animais saudáveis e enfermos, comparando elas a outros métodos para obter resultados com maior validez. Com isso, a aplicação direcionará o usuário sobre a situação do bovino baseada na teoria dos testes de diagnóstico e estatística médica, aonde se provará verdadeiramente a eficácia de nossa aplicação. Em vista de alcançar esse objetivo geral, foram especificados tais objetivos específicos:

  • Encontrar uma câmera térmica, devido a dependência para o funcionamento do projeto;
  • Especificar a amostragem de dados da IA para a base das DRB;
  • Verificar a usabilidade da linguagem Java para o projeto;
  • Analisar a viabilidade prática para com boiadeiros;
  • Confirmar a efetividade do projeto;
  • Validar o sistema através de experimentação e simulação;
